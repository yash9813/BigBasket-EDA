# 🛒 BigBasket Product Analysis - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on BigBasket product data to uncover insights related to product pricing, discounts, ratings, categories, and brands.

The objective is to demonstrate data cleaning, feature engineering, visualization, and business insight generation using Python and popular data analysis libraries.

---

## 🎯 Project Objectives

* Understand the structure and quality of the dataset.
* Handle missing values and duplicate records.
* Create new business metrics using feature engineering.
* Analyze pricing and discount patterns.
* Explore relationships between discounts, ratings, and product categories.
* Generate actionable business insights and recommendations.

---

## 🛠️ Technologies Used

## 🔍 Analysis Performed

### Data Cleaning

* Missing value treatment
* Duplicate value detection
* Data quality checks

### Feature Engineering

Created new features:

* Discount Amount
* Discount Percentage

### Univariate Analysis

* Category distribution
* Product rating distribution
* Sale price distribution
* Discount percentage distribution

### Outlier Analysis

* Boxplot visualization
* IQR-based outlier detection

### Bivariate Analysis

* Discount vs Rating
* Price vs Rating
* Category-wise analysis

### Correlation Analysis

* Correlation heatmap
* Relationship between numerical variables

### Business Insights

* Category-wise average prices
* Category-wise average discounts
* Brand-wise discount analysis
* Top-rated products
* Most expensive products

---

## 📊 Key Findings

* Certain categories have significantly higher average prices than others.
* Discount percentages vary considerably across categories and brands.
* Most products fall within a moderate discount range.
* Several price outliers were identified using the IQR method.
* Discounts show a relatively weak relationship with product ratings.

---

## 💡 Business Recommendations

* Promote highly rated products through targeted marketing campaigns.
* Review categories with exceptionally high discounts to maintain profitability.
* Analyze premium-priced products to justify pricing strategies.
* Use category-level insights to optimize promotional efforts.
